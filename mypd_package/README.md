# 产品优势
## 跨平台
- pc端依赖 utools。(你也可以尝试使用electron打包，本项目是完全基于h5的没有用到环境依赖)
- 手机端,使用Hbuilderx进行打包.支持ios,android
## 数据安全性
1. 将用户所有数据存储至用户自己的坚果云。保证隐私。
2. 坚果云拥有历史查看功能，你可以尝试恢复自己的历史数据！！！
3. 删除数据都做到了警告提示！放置误删！
